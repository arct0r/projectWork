    Workers - Hazard for the eyes

    Local effects

    Hazard assessment conclusion
    high hazard (no threshold derived)
    Explanation for hazard conclusion
    [Not publishable]

    Additional information - workers

    No hazards have been identified other than eye irritation. Eye irritation is a local, acute effect but not a parameter which is used for DNEL derivation, but for a safety instruction to wear safety goggles when performing handling activities in a dusty environment.

    Oral and dermal DNEL's have been derived from acute toxicity tests (LD50), with a conservative assessment factor.

    Inhalation DNEL's are not calculated, but set on the general value for nuisance dust (10 mg/m3).

    Additional information - General Population

    No hazards have been identified other than eye irritation. Eye irritation is a local, acute effect but not a parameter which is used for DNEL derivation, but for a safety instruction to wear safety goggles when performing handling activities in a dusty environment. This scenario is not expected for the general population.

    Oral and dermal DNEL's have been derived from acute toxicity tests (LD50), with a conservative assessment factor.

    Inhalation DNEL's are not calculated, but set on the general value for nuisance dust (10 mg/m3).
